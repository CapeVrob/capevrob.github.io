# Wedding Invite (React + Vite)

This project is set up as a React app using Vite, ready for a static responsive wedding invite website.

## Run locally

1. Install Node.js 18+ (includes `npm`)
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

4. Build production files:

```bash
npm run build
```
